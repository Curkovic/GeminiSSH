# Contributing to GeminiSSH

Thanks for checking out GeminiSSH! This is a simple AI-powered SSH assistant for Raspberry Pi and Linux systems. I’d love community ideas to improve it while keeping it lightweight and easy to set up.

## How to Contribute
1. **Fork the Repository**: Click "Fork" on GitHub to create your own copy.
2. **Make Changes**: Add features (e.g., error handling, new metrics like CPU temperature) or fix bugs.
3. **Submit a Pull Request (PR)**: Push your changes to your fork and create a PR with a brief description of what you did.
4. **Report Issues**: Use the Issues tab to suggest ideas or report problems.

## Guidelines
- Keep changes simple to maintain the <10-minute setup goal.
- Test on a Raspberry Pi or Debian-based system.
- Example ideas: better JSON parsing (e.g., with `jq`), Pi-specific metrics, or alternative AI APIs.

Let’s make SSH more fun together!
